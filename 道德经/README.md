# 道德经

<font size="4">

《道德经》，又称《老子》《道德真经》《老子五千文》，共 **81 章**，5000 余言，分上下两篇。传为春秋时期 **老子（李耳，字聃）** 所著，是道家哲学思想的奠基之作，也是中国历史上首部完整的哲学著作。

据联合国教科文组织统计，《道德经》是除《圣经》以外被译成外国文字发行量最多的文化名著。

---

## 作者

老子，姓李名耳，字聃，一字伯阳，春秋时期楚国苦县厉乡曲仁里人（今河南鹿邑）。曾担任周朝「守藏室之史」（国家图书馆馆长），后见周室衰微，骑青牛西出函谷关。关令尹喜恳请他著书传世，于是老子写下五千余言而去，「莫知其所终」。

孔子曾问礼于老子，事后对弟子感叹：
> 「鸟，吾知其能飞；鱼，吾知其能游；兽，吾知其能走……至于龙，吾不能知，其乘风云而上天。吾今日见老子，其犹龙邪！」

老子被后世尊为**道家学派创始人**，与庄子并称「老庄」。在道教中被奉为道祖，称「太上老君」，唐代追认其为李姓始祖。

---

## 成书背景

春秋末期，周王室衰微，诸侯争霸，战乱不休。铁器与牛耕的普及冲击了井田制，私田大量出现，手工业和商业蓬勃发展，但也带来贫富分化加剧、贵族横征暴敛、百姓生活困苦的局面。

老子身为周室史官，目睹王朝兴衰与民间疾苦，遂著书上下篇，言道德之意，提出了「无为而治」「道法自然」等治国安民的思想主张。

---

## 核心思想

| 概念 | 含义 |
|---|---|
| **道** | 天地万物之始之母，无形无名、不可言说，却又蕴含着一切有形事物生成发展的玄机。「道」字在书中出现 73 次 |
| **德** | 「道」在伦常领域的体现。「道」为体，「德」为用，道生万物而德畜之 |
| **自然** | 道的基本运行法则——「人法地，地法天，天法道，道法自然」 |
| **无为** | 不是什么都不做，而是不妄为、不强行干预，顺应自然之道 |
| **柔弱** | 「天下之至柔，驰骋天下之至坚」「柔弱胜刚强」 |
| **不争** | 「夫唯不争，故天下莫能与之争」 |
| **知足** | 「知足者富」「知足不辱，知止不殆，可以长久」 |
| **反者道之动** | 物极必反，对立面相互转化，这是道的运动规律 |

### 哲学层面

「道生一，一生二，二生三，三生万物」——从宇宙本源到万物的生成逻辑。阴阳对立统一是万物的本质体现，物极必反是万物演化的规律。

### 伦理层面

主张纯朴、无私、清静、谦让、贵柔、守弱、淡泊等因循自然的德性。「上善若水，水善利万物而不争。」

### 政治层面

主张对内「无为而治」，不生事扰民；对外和平共处，反对战争。「治大国若烹小鲜。」「以道佐人主者，不以兵强天下。」

---

## 篇章结构

全书共 **81 章**，分上下两篇：

| 篇名 | 章数 | 核心主题 |
|---|---|---|
| **道经** | 第 1–37 章 | 论「道」的本体、特性、宇宙生成、得道者的表现、求道的途径 |
| **德经** | 第 38–81 章 | 论「德」的运用——修身、治国、用兵、养生之道，多为政治哲学 |

> 注意：帛书本《德经》在前、《道经》在后，与通行本王弼本顺序相反。

## 历史影响

### 在中国

- **哲学**：老子建立的道生万物、道法自然的宇宙观，取代了殷周天命神学，为中国学术思想奠定了哲学基础
- **道教**：被道教奉为最高经典，与《庄子》《列子》合称「三真经」。道教的核心教义（上善若水、尊道贵德、自然无为、清静寡欲、柔弱不争、长生久视）皆源出于此
- **政治**：汉初「黄老之治」以清静无为为国策；历代明君多有取其治国之智
- **军事**：「以正治国，以奇用兵」，对后世兵家有深远影响
- **文学**：言简意深，充满诗意和哲理，是中国最精粹的哲理诗

### 在西方

《道德经》是被翻译最多的中文经典之一。德国哲学家雅斯贝尔斯将老子列为「轴心时代」核心思想家。海德格尔曾与学者合作翻译《道德经》，其「存在」哲学与老子的「道」有深刻共鸣。
