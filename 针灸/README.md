# 针灸大成

<font size="4">

《针灸大成》，又名《针灸大全》，10卷。明代**杨继洲**（济时）撰，靳贤校正，刊于万历二十九年（1601年）。

杨氏根据家传《卫生针灸玄机秘要》，参考《内经》《难经》《医经小学》《针灸聚英》《神应经》《医学入门》等明以前 20 余种针灸学著作，并结合自己针灸临床经验编成此书。全书较全面地总结了明代以前针灸学的经验和成就，是学习研究针灸最重要的参考著作之一。

## 杨继洲简介

杨继洲，又名济时，约1522—1620年，三衢（今浙江衢州）人。幼年习儒，后弃儒习医。因祖上收藏医籍甚多，经多年研习颇有心得。曾任职明世宗侍医、太医院医官。

万历年间，巡按山西监察御史赵文炳患痿痹之疾，多方医治无效，后请杨继洲诊治，**三针而愈**。赵文炳感其恩，愿资助杨氏将所著付梓刊行。杨继洲在靳贤协助下，参考20余种医籍，汇编成《针灸大成》。

## 主要贡献

1. **主张针灸与药物配合运用**：指出「疾在肠胃，非药饵不能以济；在血脉，非针刺不能以及」，强调因病制宜
2. **总结补泻手法**：汇集《内经》《难经》《神应经》及各家补泻手法，附杨氏自创手法
3. **考定腧穴名称和部位**：对361个经穴进行了系统整理
4. **收录大量针灸歌赋**：是古代针灸专书中辑录歌赋最全的
5. **保存《陈氏小儿按摩经》**：现存最早的小儿按摩专著，赖此书转载得以流传
6. **收录临床医案**：杨氏31则针灸医案，理论与实践结合
