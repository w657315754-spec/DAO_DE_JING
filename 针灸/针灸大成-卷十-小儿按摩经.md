## 一、《陈氏小儿按摩经》概述

### 历史价值

《小儿按摩经》，四明陈氏著集，原书已佚。杨继洲将其全文收录于《针灸大成》卷十，使其成为**现存最早的小儿推拿专著**。

## 二、小儿按摩基本手法

### 推拿八法

| 手法 | 操作 | 作用 |
|---|---|---|
| 推法 | 以拇指或食中二指直推 | 开通经络 |
| 运法 | 以拇指作环形推动 | 和气血 |
| 揉法 | 以指端或鱼际按揉 | 散瘀消肿 |
| 掐法 | 以指甲按掐 | 镇惊醒神 |
| 按法 | 以指端按压 | 通经止痛 |
| 摩法 | 以掌面或四指面抚摩 | 宽胸理气 |
| 搓法 | 双手对搓 | 舒筋活络 |
| 摇法 | 环转摇动关节 | 滑利关节 |

### 五脏与五指对应

> 小儿按摩以手掌五指对应五脏，是核心理论。

| 手指 | 对应脏 | 推拿部位 | 主治 |
|---|---|---|---|
| 拇指 | **脾** | 拇指螺纹面（脾经） | 消化不良、呕吐腹泻 |
| 食指 | **肝** | 食指螺纹面（肝经） | 惊风、烦躁 |
| 中指 | **心** | 中指螺纹面（心经） | 高热、夜啼 |
| 无名指 | **肺** | 无名指螺纹面（肺经） | 咳嗽、感冒 |
| 小指 | **肾** | 小指螺纹面（肾经） | 遗尿、发育不良 |

### 推法方向

- **向心为补，离心为泻**
- 脾经宜补不宜泻
- 肝经宜泻不宜补
- 心经宜清不宜补

## 四、小儿望诊与面部五位图

### 面部五位图

> 小儿面部分为五个区域，各对应五脏。

| 部位 | 对应 | 正常色 | 病色 |
|---|---|---|---|
| 额（天庭） | **心** | 红润 | 赤（心热） |
| 鼻（年寿） | **脾** | 黄润 | 黄（脾虚） |
| 左颊 | **肝** | 青润 | 青（惊风） |
| 右颊 | **肺** | 白润 | 白（肺寒） |
| 下巴（地阁） | **肾** | 黑润 | 黑（肾虚） |

### 察色验病诀

- 面青：主惊风、主痛
- 面赤：主热、主痰
- 面黄：主脾虚、主湿
- 面白：主虚寒、主肺疾
- 面黑：主肾虚、主寒痛

## 五、小儿针灸法（卷十附录）

### 小儿常用针灸穴

| 穴名 | 主治 | 操作 |
|---|---|---|
| 百会 | 惊风、囟门迟闭 | 灸三壮 |
| 人中 | 惊厥昏迷 | 刺出血 |
| 十宣 | 高热惊厥 | 刺出血 |
| 四缝 | 疳积、消化不良 | 刺出黄水（挑疳法） |
| 神阙 | 腹泻虚脱 | 隔盐灸 |
| 身柱 | 小儿发育、虚弱 | 灸七七壮 |

### 小儿针灸禁忌

> 小儿的针灸须更加谨慎。

- 囟门未合者**禁针百会**
- 小儿皮肤娇嫩，**灸不用大炷**
- 小儿脏气清灵，**不宜深刺**
- 腹泻脱水者**宜灸不宜针**

## 六、《陈氏小儿按摩经》的现代价值

《小儿按摩经》所载手法至今仍广泛应用于小儿推拿临床：

1. **绿色无药**：推拿不伤脾胃，是小儿最安全的疗法
2. **易于操作**：手法简单，家长可以学习家用
3. **效果确切**：对感冒、发热、咳嗽、腹泻、夜啼等小儿常见病效果良好
4. **理论完整**：建立了五指-五脏-五色的完整理论体系
