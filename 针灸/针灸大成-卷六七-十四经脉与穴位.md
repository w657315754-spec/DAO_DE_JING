## 卷六·手太阴肺经（11穴）

**循行路线**：中焦 → 大肠 → 贲门 → 膈肌 → 肺 → 腋下 → 上肢内侧前缘 → 拇指桡侧

| 穴名 | 定位 | 主治 |
|---|---|---|
| 中府 | 胸前壁外上方，平第一肋间隙 | 咳嗽、气喘、胸痛 |
| 云门 | 锁骨下窝凹陷处 | 同上 |
| 天府 | 腋前纹头下3寸 | 鼻衄、瘿气 |
| 侠白 | 天府下1寸 | 咳嗽、心痛 |
| **尺泽** | 肘横纹肱二头肌腱桡侧 | ⭐ 清肺热、咳喘 |
| 孔最 | 尺泽下5寸 | 咳血、痔血 |
| **列缺** | 腕上1.5寸 | ⭐ 偏头痛、感冒、头项 |
| 经渠 | 腕横纹上1寸 | 咳嗽、腕痛 |
| **太渊** | 腕横纹桡动脉旁 | ⭐ 无脉症、咳嗽（脉会） |
| 鱼际 | 第一掌骨中点赤白肉际 | 咽喉痛、发热 |
| **少商** | 拇指桡侧指甲角 | ⭐ 急救、咽喉肿痛、高热 |

## 卷七·督脉（28穴）

> 后正中线，阳脉之海。
