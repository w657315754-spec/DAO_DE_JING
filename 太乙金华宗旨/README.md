# 太乙金华宗旨

<font size="4">

《太乙金华宗旨》，又名《金华宗旨》《长生术》，共 **13 章**。托名唐代 **吕洞宾**（纯阳吕祖）降坛著述，为明清时期扶乩产物的道家内丹修炼经典。

与《唱道真言》《道乡集》《大成捷要》《古书隐楼藏书》并列为明清道家经典著作的代表。

---

## 历史背景

清康熙戊申年（1668年），吕祖奉敕偕邱、谭二真人降坛于毗陵（今常州）白龙精舍，向潘乾德等七人宣扬宗旨。后因人事变迁，康熙壬申年（1692年）由孝悌王重提宗旨，传授屠乾元、张坎真等七人。由张坎真订辑成书。

《太乙金华宗旨》为**净明法派**所传，不在南北两宗，乃吕祖代许旌阳真人阐述宗旨。

---

## 版本差异

本书存在两个主要版本系统：

| 系统 | 代表版本 | 特点 |
|---|---|---|
| **净明派** | 《道藏辑要》本（邵志琳订） | 13章，首章较简，流传最广 |
| **龙门派** | 《道藏续编》本（闵一得订） | 13章，首章有慧真子注，更详细 |

---

## 西方的巨大影响

1929年，德国汉学家 **卫礼贤**（Richard Wilhelm）将前八章翻译为德文，题为《金花的秘密》（Das Geheimnis der Goldenen Blüte）。心理学大师 **荣格**（C.G. Jung）为该书写了长篇评注。

荣格从中发现了与他的「集体无意识」「曼陀罗」「自性化」理论的惊人契合，称之为来自东方的「心灵科学」。该书出版后轰动欧洲，成为畅销书，后翻译为英文、日文等多国语言。

## 核心概念

| 概念 | 含义 |
|---|---|
| **太乙** | 无上、最初、元始 |
| **金华** | 即光、金丹。取象于金色之花，秘一「光」字在内 |
| **天心** | 日月中，两眉之间。儒曰虚中，释曰灵台，道曰玄关 |
| **回光** | 逆法：两目内视天心，使光回返，不走泄于外 |
| **元神** | 先天真性，超生死、越阴阳。魂之所藏 |
| **识神** | 后天分别心，逐物而动。魄之所附 |

## 各章概要

| 章 | 篇名 | 核心内容 |
|---|---|---|
| 一 | **天心** | 自然曰道。金华即光。回光注想天心。 |
| 二 | **元神识神** | 元神超生死，识神逐物欲。消魄全魂。 |
| 三 | **回光守中** | 逆法回光。圣圣相传不离返照。止观双修。 |
| 四 | **回光调息** | 心息相依，以听息入静，治昏沉散乱。 |
| 五 | **回光差谬** | 无记空、蕴界、万缘随逐——三大歧途。 |
| 六 | **回光证验** | 金华乍吐 → 正放 → 大凝。三阶段验证。 |
| 七 | **回光活法** | 日常应事接物中亦可行回光，不废正业。 |
| 八 | **逍遥诀** | 「凝神入炁穴」为金丹密钥。三节功夫。 |
| 九 | **百日立基** | 百日专功方有真光。回风混合，百日功灵。 |
| 十 | **性光识光** | 本性光与识神光之别。 |
| 十一 | **坎离交媾** | 坎离即阴阳。七窍内返为坎，外走为离。 |
| 十二 | **周天** | 一日一周天，一刻一周天。活子时之妙。 |
| 十三 | **劝世歌** | 以歌诀形式总结全书，劝人修行。 |
