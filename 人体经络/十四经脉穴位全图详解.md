# 十四经脉穴位全图详解

<font size="4">

人体全身经络穴位以十四经（十二正经 + 任督二脉）为核心，共 **361 个经穴**。以下按手三阴、手三阳、足三阴、足三阳、任脉、督脉系统整理，附循行路线、穴位清单、常用穴定位与主治。

> 📸 **经络穴位图片参考**：[中医宝典 — 人体穴位图](http://zhongyibaodian.net/a/13452.html) | [经穴穴位图14张全套](http://zhongyibaodian.net/a/137714.html)

## 一、手三阴经（上肢内侧）

### 1. 手太阴肺经（11穴）

**循行路线**：中焦 → 大肠 → 贲门 → 膈肌 → 肺 → 腋下 → 上肢内侧前缘 → 拇指桡侧

**穴位**：中府、云门、天府、侠白、尺泽、孔最、列缺、经渠、太渊、鱼际、少商

| 常用穴 | 定位 | 主治 |
|---|---|---|
| 尺泽 | 肘横纹肱二头肌腱桡侧 | 清肺热、止咳喘 |
| 列缺 | 腕横纹上1.5寸 | 偏头痛、落枕、感冒 |
| 太渊 | 腕横纹桡动脉旁 | 补气、止咳、脉管病 |
| 少商 | 拇指桡侧指甲角 | 咽喉肿痛、急救、高热 |

---

### 2. 手厥阴心包经（9穴）

**循行路线**：胸 → 上肢内侧中线 → 中指（支脉至无名指）

**穴位**：天池、天泉、曲泽、郄门、间使、内关、大陵、劳宫、中冲

| 常用穴 | 定位 | 主治 |
|---|---|---|
| 内关 | 腕上2寸两筋间 | 心痛、心悸、呕吐、失眠 |
| 劳宫 | 掌心 | 中暑、中风昏迷、口疮 |

---

### 3. 手少阴心经（9穴）

**循行路线**：心中 → 腋下 → 上肢内侧后缘 → 小指桡侧

**穴位**：极泉、青灵、少海、灵道、通里、阴郄、神门、少府、少冲

| 常用穴 | 定位 | 主治 |
|---|---|---|
| 神门 | 腕横纹尺侧端 | 安神、失眠、健忘、心悸 |
| 少冲 | 小指桡侧指甲角 | 急救、心悸、发热 |

---

## 二、手三阳经（上肢外侧）

### 1. 手阳明大肠经（20穴）

**循行路线**：食指桡侧 → 上肢外侧前缘 → 肩 → 缺盆（联络肺脏 → 大肠）→ 颈部 → 面颊 → 鼻旁

**穴位**：商阳、二间、三间、合谷、阳溪、偏历、温溜、下廉、上廉、手三里、曲池、肘髎、手五里、臂臑、肩髃、巨骨、天鼎、扶突、口禾髎、迎香

| 常用穴 | 定位 | 主治 |
|---|---|---|
| 合谷 | 手背1-2掌骨间 | 头痛、牙痛、面瘫、发热（止痛要穴） |
| 曲池 | 肘横纹外侧端 | 降压、清热、止痒、肩肘痛 |
| 迎香 | 鼻翼旁 | 鼻塞、鼻炎、面瘫 |

---

### 2. 手少阳三焦经（23穴）

**循行路线**：无名指 → 上肢外侧中线 → 肩 → 缺盆（络心包 → 三焦）→ 耳后 → 耳上 → 面颊 → 眉梢

**穴位**：关冲、液门、中渚、阳池、外关、支沟、会宗、三阳络、四渎、天井、清冷渊、消泺、臑会、肩髎、天髎、天牖、翳风、瘈脉、颅息、角孙、耳门、和髎、丝竹空

| 常用穴 | 定位 | 主治 |
|---|---|---|
| 外关 | 腕背2寸 | 偏头痛、耳鸣、感冒、落枕 |
| 支沟 | 腕背3寸 | 便秘、胁痛、耳鸣 |
| 翳风 | 耳后乳突前 | 面瘫、耳鸣、腮腺炎 |

---

### 3. 手太阳小肠经（19穴）

**循行路线**：小指尺侧 → 上肢外侧后缘 → 肩胛 → 大椎 → 缺盆（络心 → 小肠）→ 颈 → 面颊 → 耳前

**穴位**：少泽、前谷、后溪、腕骨、阳谷、养老、支正、小海、肩贞、臑俞、天宗、秉风、曲垣、肩外俞、肩中俞、天窗、天容、颧髎、听宫

| 常用穴 | 定位 | 主治 |
|---|---|---|
| 后溪 | 第5掌指关节后 | 颈椎病、落枕、急性腰扭伤 |
| 天宗 | 肩胛冈下窝 | 肩痛、乳痈、气喘 |
| 听宫 | 耳屏前张口凹陷 | 耳鸣、耳聋、牙痛 |

---

## 三、足三阳经

### 1. 足阳明胃经（45穴）

**循行路线**：鼻旁 → 鼻根 → 上齿龈 → 唇（环绕）→ 下颌 → 耳前 → 前额 → 颈 → 锁骨上窝 → 膈肌 → 胃 → 下肢外侧前缘 → 第二趾外侧

**穴位**：承泣、四白、巨髎、地仓、大迎、颊车、下关、头维、人迎、水突、气舍、缺盆、气户、库房、屋翳、膺窗、乳中、乳根、不容、承满、梁门、关门、太乙、滑肉门、天枢、外陵、大巨、水道、归来、气冲、髀关、伏兔、阴市、梁丘、犊鼻、足三里、上巨虚、条口、下巨虚、丰隆、解溪、冲阳、陷谷、内庭、厉兑

| 常用穴 | 定位 | 主治 |
|---|---|---|
| 足三里 | 犊鼻下3寸 | 强壮要穴、脾胃病（保健第一要穴） |
| 天枢 | 脐旁2寸 | 便秘、腹泻、肠胃病 |
| 丰隆 | 外踝上8寸 | 化痰要穴、咳嗽、肥胖、眩晕 |
| 内庭 | 足背2-3趾缝 | 胃火牙痛、口臭、便秘 |

---

### 2. 足少阳胆经（44穴）

**循行路线**：目外眦 → 头侧 → 耳后 → 肩 → 缺盆（络肝 → 胆）→ 胸侧 → 下肢外侧中线 → 第四趾外侧

**穴位**：瞳子髎、听会、上关、颔厌、悬颅、悬厘、曲鬓、率谷、天冲、浮白、头窍阴、完骨、本神、阳白、头临泣、目窗、正营、承灵、脑空、风池、肩井、渊腋、辄筋、日月、京门、带脉、五枢、维道、居髎、环跳、风市、中渎、膝阳关、阳陵泉、阳交、外丘、光明、阳辅、悬钟、丘墟、足临泣、地五会、侠溪、足窍阴

| 常用穴 | 定位 | 主治 |
|---|---|---|
| 风池 | 枕后凹陷 | 头痛、感冒、颈椎病、醒脑 |
| 肩井 | 肩上凹陷 | 肩痛、乳腺胀痛、落枕 |
| 环跳 | 臀外侧 | 坐骨神经痛、腰腿痛 |

---

### 3. 足太阳膀胱经（67穴）

**循行路线**：目内眦 → 头顶 → 项后脊柱两侧 → 腰背 → 下肢外侧后缘 → 小趾外侧

**穴位**：睛明、攒竹、眉冲、曲差、五处、承光、通天、络却、玉枕、天柱、大杼、风门、肺俞、厥阴俞、心俞、督俞、膈俞、肝俞、胆俞、脾俞、胃俞、三焦俞、肾俞、气海俞、大肠俞、关元俞、小肠俞、膀胱俞、中膂俞、白环俞、上髎、次髎、中髎、下髎、会阳、承扶、殷门、浮郄、委阳、委中、附分、魄户、膏肓、神堂、譩譆、膈关、魂门、阳纲、意舍、胃仓、肓门、志室、胞肓、秩边、合阳、承筋、承山、飞扬、跗阳、昆仑、仆参、申脉、金门、京骨、束骨、足通谷、至阴

| 常用穴 | 定位 | 主治 |
|---|---|---|
| 肾俞 | 第2腰椎棘突下旁开1.5寸 | 腰痛、肾病、耳鸣、强壮 |
| 委中 | 腘横纹中点 | 腰背痛（腰背委中求） |
| 承山 | 腓肠肌两肌腹间 | 腿抽筋、痔疮 |
| 至阴 | 小趾外侧指甲角 | 胎位不正 |

## 四、足三阴经

### 1. 足太阴脾经（21穴）

**循行路线**：大趾内侧 → 下肢内侧中间（内踝上8寸以下行前缘） → 腹部 → 膈肌 → 食道 → 舌下

**穴位**：隐白、大都、太白、公孙、商丘、三阴交、漏谷、地机、阴陵泉、血海、箕门、冲门、府舍、腹结、大横、腹哀、食窦、天溪、胸乡、周荣、大包

| 常用穴 | 定位 | 主治 |
|---|---|---|
| 三阴交 | 内踝上3寸 | 妇科要穴、脾胃、肝肾（三条阴经交会） |
| 阴陵泉 | 胫骨内侧髁下 | 水肿、小便不利 |
| 血海 | 髌底内侧端上2寸 | 月经不调、皮肤瘙痒 |

---

### 2. 足厥阴肝经（14穴）

**循行路线**：大趾外侧 → 下肢内侧前缘 → 环阴器 → 腹部 → 膈肌 → 胁肋 → 咽喉 → 目 → 头顶

**穴位**：大敦、行间、太冲、中封、蠡沟、中都、膝关、曲泉、阴包、足五里、阴廉、急脉、章门、期门

| 常用穴 | 定位 | 主治 |
|---|---|---|
| 太冲 | 足背1-2跖骨间 | 头痛、眩晕、胁痛、降压 |
| 期门 | 乳下第6肋间 | 胁痛、腹胀、乳腺病 |

---

### 3. 足少阴肾经（27穴）

**循行路线**：小趾下 → 足心 → 内踝后 → 下肢内侧后缘 → 脊柱 → 肾 → 膀胱 → 肝 → 膈肌 → 肺 → 咽喉 → 舌根

**穴位**：涌泉、然谷、太溪、大钟、水泉、照海、复溜、交信、筑宾、阴谷、横骨、大赫、气穴、四满、中注、肓俞、商曲、石关、阴都、腹通谷、幽门、步廊、神封、灵墟、神藏、彧中、俞府

| 常用穴 | 定位 | 主治 |
|---|---|---|
| 涌泉 | 足底前1/3凹陷 | 眩晕、失眠、急救 |
| 太溪 | 内踝与跟腱间 | 补肾要穴、腰痛、耳鸣 |

---

## 五、任督二脉

### 1. 任脉（阴脉之海，24穴）

**循行路线**：小腹 → 会阴 → 前正中线 → 咽喉 → 下唇 → 环绕口唇 → 目下

**穴位**：会阴、曲骨、中极、关元、石门、气海、阴交、神阙、水分、下脘、建里、中脘、上脘、巨阙、鸠尾、中庭、膻中、玉堂、紫宫、华盖、璇玑、天突、廉泉、承浆

| 常用穴 | 定位 | 主治 |
|---|---|---|
| 关元 | 脐下3寸 | 强壮要穴、补肾固元 |
| 气海 | 脐下1.5寸 | 补气要穴、腹胀 |
| 神阙 | 肚脐中央 | 腹痛、脱肛（宜灸不宜针） |
| 中脘 | 脐上4寸 | 胃病要穴、消化不良 |
| 膻中 | 两乳头连线中点 | 胸闷、气喘、缺乳（气会膻中） |

---

### 2. 督脉（阳脉之海，28穴）

**循行路线**：小腹 → 会阴 → 脊柱内 → 项 → 头顶 → 前额 → 鼻柱

**穴位**：长强、腰俞、腰阳关、命门、悬枢、脊中、中枢、筋缩、至阳、灵台、神道、身柱、陶道、大椎、哑门、风府、脑户、强间、后顶、百会、前顶、囟会、上星、神庭、印堂、素髎、水沟（人中）、兑端、龈交

| 常用穴 | 定位 | 主治 |
|---|---|---|
| 命门 | 第2腰椎棘突下 | 补肾壮阳、腰痛 |
| 大椎 | 第7颈椎棘突下 | 退热要穴、颈椎病 |
| 百会 | 两耳尖连线中点 | 头痛、眩晕、升阳 |
| 人中 | 人中沟上1/3处 | 急救要穴、昏迷 |

## 六、奇经八脉

除任督二脉外，还有以下六条奇经：

| 经脉 | 循行特点 |
|---|---|
| 冲脉 | 起于小腹，上行胸中，为血海 |
| 带脉 | 环腰一周，约束诸经 |
| 阴跷脉 | 起于足跟内侧，上行目内眦 |
| 阳跷脉 | 起于足跟外侧，上行目内眦 |
| 阴维脉 | 维系诸阴经 |
| 阳维脉 | 维系诸阳经 |
