# 人体经络

<font size="4">

人体经络系统是中医理论的核心组成部分，主要包括**十四经脉**（十二正经 + 任督二脉），共 **361 个经穴**。

## 十四经脉一览

### 手三阴经（上肢内侧）
| 经脉 | 穴位数 | 循行路线 |
|---|---|---|
| 手太阴肺经 | 11穴 | 胸 → 上肢内侧前缘 → 拇指 |
| 手厥阴心包经 | 9穴 | 胸 → 上肢内侧中线 → 中指 |
| 手少阴心经 | 9穴 | 腋下 → 上肢内侧后缘 → 小指 |

### 手三阳经（上肢外侧）
| 经脉 | 穴位数 | 循行路线 |
|---|---|---|
| 手阳明大肠经 | 20穴 | 食指 → 上肢外侧前缘 → 肩 → 鼻旁 |
| 手少阳三焦经 | 23穴 | 无名指 → 上肢外侧中线 → 肩 → 耳 → 眉梢 |
| 手太阳小肠经 | 19穴 | 小指 → 上肢外侧后缘 → 肩胛 → 耳前 |

### 足三阳经（下肢）
| 经脉 | 穴位数 | 循行路线 |

### 足三阴经（下肢）
| 经脉 | 穴位数 | 循行路线 |

## 经络养生基本原则

1. **春养肝**（省酸增甘）— 足厥阴肝经
2. **夏养心**（省苦增辛）— 手少阴心经
3. **秋养肺**（省辛增酸）— 手太阴肺经
4. **冬养肾**（省咸增苦）— 足少阴肾经
5. **四季末各18日养脾**（省甘增咸）— 足太阴脾经
