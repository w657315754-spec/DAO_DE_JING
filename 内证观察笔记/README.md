## 内容简介

中医与西医的根本不同之处在于，中医除了治疗人的肉体，还治疗人的**精、气、神**。其精气神部分，比如经络穴位、五运六气、五脏六腑，便是与西医完全不同的概念，这些，都是中医「**内证**」的领域。

该书不仅仅是一本谈人体奥秘的书，它所揭示的是人的生命与宇宙交流的独特方式和通道，生命的运行与大自然的神秘关联。书中以道教文化和知识为背景，将中医的脏腑、经络等放在人体生命与宇宙自然的关系中进行说解描画。

## 核心概念

| 概念 | 含义 |
|---|---|
| **内证** | 一种内在的实证功夫，通过自己的身体去探索、观察及求知的科学方法。有了内证才有中医经络之学，百草药效才得以验证 |
| **五眼** | 肉眼、天眼、慧眼、法眼、佛眼——内证观察的不同层次 |
| **太极器官** | 人体中天然存在的太极形态的能量结构，如丹田、穴位、脏腑等 |
| **性命之学** | 中国古代的生命科学，涵盖「性」（先天元神）与「命」（后天形体）两个维度 |
| **三界宇宙** | 将宇宙分为欲界、色界、无色界三个层次，对应人体的精、气、神 |
| **五运六气** | 天地运行之数与人体气化的对应关系 |

## 全书结构

全书分为**上编、中编、下编**三大部分：

### 上编 · 内证漫谈

从理论上探讨什么是内证、内证的历史、内证的方法论。核心内容包括：

- **讲述者说**：作者自述写作缘起、方法论与世界观
- **寻找内证**：内证的定义、内证「五眼」、内证实验室的概念
- **性命之学**：性的本质、命的本质、元神与识神、天人合一
- **大宇宙**：三垣、七政、二十八星宿与人体对应的关系
- **阴阳、五运六气与五行**：宇宙间活生生的阴阳物质
- **宇宙与疾病的游戏**：疾病与宇宙力量的关系
- **中药与西药**：两者在体内的不同运作方式

### 中编 · 神奇的太极器官与五藏

记录作者通过内证观察到的五脏与太极器官。核心内容包括：

- **神奇系统**：元神、三魂、七魄的实证观察
- **太极器官**：人体内天然存在的太极能量结构
- **五脏详解**：心藏、肝藏、脾藏、肺藏、肾藏的内证观察实录

### 下编 · 十二正经观察

逐一记录十二正经的经络循行内证观察：

- 经络管道与穴位中的太极器官
- 三焦经、胆经、肝经、肺经、大肠经、胃经
- 脾经、心经、小肠经、膀胱经、肾经、心包经

## 为何重要

本书是当代极少数敢于公开记录「内证」经验的著作。作者以亲身实证的方式，重新走过扁鹊、华佗、张仲景、孙思邈等历代大医家的内证之路，用文字还原出一个被现代人遗忘的人体能量世界。

在中医与道教的交叉领域，本书具有独特的价值：

1. **实证性**：不是空谈理论，而是具体的内证观察记录
2. **系统性**：从五脏到经络、从太极器官到宇宙星宿，体系完整
3. **当代性**：用现代人能理解的语言讲述古老的内证学问
4. **桥梁性**：在中医、道教、传统文化与现代科学之间搭建沟通的桥梁

## 推荐序言

**刘力红**（《思考中医》作者）评价：

> 「人体是内证实验室。」

《思考中医》作者刘力红老师为本书作序力荐，认为内证是中医学理得以建立的实证基础。没有内证，就没有经络之说；没有内证，就没有百草药效的确认。
